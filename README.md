# NullifierCore - Telegram Bot для выгрузки подписчиков каналов

NullifierCore v1.2.0 "Unique Subscribers" - это бот для Telegram, который позволяет выгружать список подписчиков из каналов, где бот является администратором.

## Особенности

- Работа с Bot API и MTProto API Telegram
- Добавление/удаление каналов через команды
- Выгрузка списка подписчиков канала в CSV файл
- Поддержка приватных каналов
- Удобный интерфейс с инлайн-кнопками
- Отображение прогресса операций
- Возможность отмены выгрузки

## Требования

- Python 3.7+
- python-telegram-bot
- telethon
- aiohttp
- python-dotenv (опционально)

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/nullifiercore-bot.git
cd nullifiercore-bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example` и заполните его вашими API-ключами:
```
API_ID=ваш_api_id
API_HASH=ваш_api_hash
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

## Запуск

Для запуска бота выполните:
```bash
python run.py
```

Для проверки подключения к API можно запустить тестовый скрипт:
```bash
python test_connection.py
```

## Использование

1. Добавьте бота администратором канала
2. Запустите бота через команду `/start`
3. Добавьте канал с помощью команды `/addchannel @username` или `/addchannel -100123456789`
4. Используйте команду `/channels` для просмотра списка добавленных каналов
5. Выберите канал для выгрузки подписчиков
6. Дождитесь завершения процесса и получите CSV-файл

## Команды

- `/start` - Начать работу с ботом
- `/channels` - Показать список добавленных каналов
- `/addchannel` - Добавить канал (формат: `/addchannel @username` или `/addchannel -100123456789`)
- `/removechannel` - Удалить канал из списка
- `/help` - Показать справку
- `/cancel` - Отменить текущую операцию

## Советы

- Для приватных каналов необходимо использовать ID канала в формате `-100xxxxxxxxx`
- Бот должен быть администратором канала для получения списка подписчиков
- Процесс выгрузки подписчиков может занять некоторое время в зависимости от размера канала
